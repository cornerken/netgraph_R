# R语言 基于共现提取《雪中悍刀行》人物关系并做网络图


基于简单共现关系，编写 R 代码从《雪中悍刀行》小说纯文本中提取出人物关系网络，并用 networkD3 将生成的网络可视化。

* 雪中悍刀行中文文本 busan.txt
* 人物名词           人物表.xlsx
* 多称谓人物表       多称谓人物.xlsx
* 主要程序           netgraphic.rmd
* 报告结果           netgraphic.html
